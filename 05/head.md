# 编辑部专版 - Brainfuck 和他的朋友们

前些天在上网冲浪的时候看到了一门叫做 Brainfuck 的编程语言，秉承着了解一门编程语言就先学会它的 Hello,World 的原则，我搜索到了 Brainfuck 的 Hello World 源代码:

```brainfuck
>+++++++++[<++++++++>-]<.>++++++[<+++++>-]<++++.+++++++..+++.>>>+++++[<+++++>-]<<.>>>++++++++[<+++++++++>-]<---.<<<<.+++.------.---------.>+.
```

这 TM 是编程语言?

## Brainfuck

Brainfuck 是一门简单的深奥语言，具有图灵完备性，它仅用了八个字符来作为指令。Brainfuck 基于一个字节数组（内存带）和指针操作，指令如下：

| 字符 | 作用                           |
| ---- | ------------------------------ |
| >    | 指针右移                       |
| <    | 指针左移                       |
| +    | 当前指针指向的字节值加 1       |
| -    | 当前指针指向的字节值减 1       |
| .    | 输出当前字节（ASCII 字符）     |
| ,    | 输入一个字节到当前指针位置     |
| [    | 若当前字节为 0，跳转到匹配的 ] |
| ]    | 若当前字节非 0，跳回匹配的 [   |

那么，之前提到的代码就好解释了，比如代码片段

```brainfuck
>+++++++++[<++++++++>-]<.
```

- > +++++++++：指针右移，当前单元格存储 9（作为循环计数器）。

- [<++++++++>-]：循环 9 次，每次左移指针并加 8（9×8=72，即 'H' 的 ASCII 值）。

- <.：指针左移并输出当前单元格的值（72 → H）。

其他的片段也同理，太麻烦了不做解释，如果你想要体验这门深奥编程语言，可以下载这个 C 语言写的[Brainfuck 解释器](https://github.com/fabianishere/brainfuck)

在看到了这个语言之后，我又找到了其他的奇怪的编程语言

## Ook

我建议各位用小米设备并循环播放 Are you OK 来使用这个语言。这东西就是 Brainfuck 的一个变种，区别就是用八种不同语气的 Ook 来替换 Brainfuck 的八个指令。

| Ook! 指令 | Brainfuck 指令 | 作用                            |
| --------- | -------------- | ------------------------------- |
| Ook. Ook? | >              | 指针右移                        |
| Ook? Ook. | <              | 指针左移                        |
| Ook. Ook. | +              | 当前单元格值加 1                |
| Ook! Ook! | -              | 当前单元格值减 1                |
| Ook! Ook. | .              | 输出当前字符（ASCII）           |
| Ook. Ook! | ,              | 输入字符到当前单元格            |
| Ook! Ook? | [              | 循环开始（若当前值为 0 则跳转） |
| Ook? Ook! | ]              | 循环结束（若当前值非 0 则跳回） |

我觉得按照这个思路，我一天能创造八门语言，比如说`Kitten No.1语言`

| Kitten No.1. Kitten No.1? | >              | 指针右移                        |
| Kitten No.1? Kitten No.1. | <              | 指针左移                        |
| Kitten No.1. Kitten No.1. | +              | 当前单元格值加 1                |
| Kitten No.1! Kitten No.1! | -              | 当前单元格值减 1                |

| Kitten No.1? Kitten No.1! | ]              | 循环结束（若当前值非 0 则跳回） |

## Piet

看下面这幅图片：
![](./assets/head-1.webp)
Piet 是一种基于颜色代码的深奥编程语言，由 David Morgan-Mar 设计。其程序以抽象艺术风格的位图图像呈现，通过指针在连续色块区域间的移动实现指令执行与编译过程

这是画作吗？不，这是一门叫做 Piet 的编程语言的代码。该语言使用包含 20 种颜色的调色板，其中 18 种颜色按 6 种色相（红、黄、绿、青、蓝、紫）和 3 级亮度（明、常、暗）构成循环系统。指令由色相与亮度的变化差值决定，数据通过堆栈结构存储。执行时通过方向指针(DP)和色块选择器(CC)控制流程，黑色区域触发转向判定直至程序终止。由于编译器不好找，我们在这里贴几张 Piet 语言的程序
